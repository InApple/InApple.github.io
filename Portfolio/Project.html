    2. 프로젝트 소개
    • 본 프로젝트는 기존의 번역기를 이용하여 문서 이미지 파일이나, 자막등의 이미지를 번역하여 실시간으로 출력하는 프로그램 제작을 목표로 함.
    주요기능
    1. 사용자가 화면 상의 일정한 구역을 지정하여 번역.
    2. 이미지상의 문자열을 추출 및 번역한 뒤 화면에 재출력.
    3. 번역되는 텍스트는 자연스러운 출력을 위해 기존 이미지의 텍스트를 지운 뒤 출력..
    4. 이미지를 읽는 방법(OCR) 및 이미지 보정은 딥러닝을 이용.
    적용기술 API 구글 혹은 네이버 번역 API
    인공지능텍스트 인식 성능을 높이기 위한 딥러닝 기술
    영상처리번역 텍스트를 자연스럽게 출력하기 위해 기존 이미지에서 텍스트를 지워 보정하는 기술
    기타윈도우 폼 디자인 등의 GUI 디자인 기술

  2. 기대효과 및 활용분야

    3. 프로그램의
    편의성     		1. 여러 가지의 프로그램 없이 이미지를 읽어 그대로 화면에 출력시키는 편의성.
    2. 화면의 모든 텍스트를 번역하는 것이 아닌 사용자가 직접 번역할 부분을 지정하여 번역함으로써 사용자 개인의 편의성을 높임.
    다양한 활용 1. 사진이나 문서등의 이미지 문서만이 아닌 브라우저 자체 번역이 불가능한 사이트나, 영화 자막등 다양한 곳에서 활용 가능.
    접근성 용이 1.단축기를 이용한 프로그램의 조작이 가능.
    2. 백그라운드 프로그램으로 한번 실행 후 사용자가 종료 할때까지 지속적으로 번역을 실행.
    3. 영화 자막이나, 게임, 원서 등 프로그램에 대한 지식이 없어도 문자열 추출이 용이.

    4. 예상 결과물 이미지설명

    PDF, 이미지 문서를 일부 지정(레이아웃 지정) 후 문서 텍스트를 번역.
